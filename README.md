# PixelGrab

**PixelGrab** is a simple color extractor tool that helps you pull colors from images quickly and cleanly.

Upload an image, grab the visible colors, and use them for design, branding, UI work, or creative inspiration.

🌐 **Live Site:** https://piixelgrab.netlify.app/

## What It Does

PixelGrab helps users:

- extract colors from images
- preview color palettes instantly
- pick useful colors for design projects
- speed up the process of finding visual inspiration

## Why I Built It

Finding the right colors from an image should be fast and painless.

PixelGrab was built to make that process easier for designers, frontend developers, and creatives who want to grab colors without dealing with bloated design software.

## Features

- image-based color extraction
- clean and simple interface
- fast palette generation
- useful for UI, branding, and visual design work

## Tech Stack

- Svelte
- JavaScript
- Tailwind CSS
